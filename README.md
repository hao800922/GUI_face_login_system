## How to install?
$<code>git clone https://github.com/hao800922/GUI_face_login_system.git</code>  
$<code>cd GUI_face_login_system.git</code>  
$<code>sudo pip3 install -r requirements.txt</code>  
    
## How to use?
$<code>python3 MAIN_MENU.py</code>
